ARTICLE 4 : DESCRIPTION OF APPLICATION FUNCTIONS

The list of features and services of the Application described below is not exhaustive and may be modified at any time without the responsibility of the MOODDY company being engaged by the User or a third party.

To date, the User will be able to access eight functionalities:

  • The "Chat" page which allows access to an instant chat space allowing two or more Users to communicate with each other. From this page the User accesses the profile of the other user, and can block, report and delete conversations if he wishes.
  • The " Rendez-Mood " page which gives access to general music news.
  • The "User Search Around Me" page that allows you to search for another user of the Application from its geolocation by digitally determining the geographical search perimeter.

Once another User has been identified, the User will be able to access:

  • to the other user's last musical activity.
  • to his profile and thus follow him via the "Follow" icon, communicate with him via the "Chat" icon, report him in case of suspected abuse via the "Report" icon, follow his musical activity, like music he likes via the "Liker" icon.

The geolocation information is updated regularly, even without refreshing the page by the User. The User can disable geolocation in the "Settings" tab if he wishes.

  • The "Blocked Users" page that allows each User to prevent another User from accessing their profile and coming into contact via the "Chat" function.
  • The "Invite by SMS" page that allows you to send a link to a third party to download the Application.
  • The "Settings" page that allows Users to modify Application settings.
  • The "Review Tutorial" button.
  • The "Logout" button to close the Application and log out.
  • The "Profile" page which allows the User to access his account in order to inform him and to change at any time the parameters of use of the Application. This page displays the information recorded by the User during the creation of his account but also the last listened and shared music, the number of messages, the persons subscribed to his account (Followers) and the accounts of the other Users to which the User subscribed (Following), the synchronization or not of the account with the other music libraries created from application or Internet site third, the access to the instant messaging (chat). It is from the profile page that the User can change his password.
  • The "Playback in progress" page which allows you to listen to music and offers a music search bar via the "magnifying glass" icon, which will be carried out according to the synchronisation chosen by the User as well as a selection of Moods (mood icons) in the form of smileys that the User can select in real time according to his mood. The selected Moods will be visible to other Users. Users via the "+" icon can add music to their Mooddy playlist and share it on their news feed or Facebook.
  • The "News Feed" page displays the music shared by the people the User is following. It allows Users to "Liker" music shared by another user, comment, listen, access the other user's profile, post a user, delete a comment, view the mood associated with the music, identify someone or post comments on each publication.

Whatever page is displayed, a "Bell" icon will be displayed to allow the User to know if he has received a notification about a new follower, a "Like", a comment or a message on the chat.

The User may at any time delete his account by sending a request to the email address contact@mooddy-app.com.
